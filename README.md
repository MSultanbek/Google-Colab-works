# 📊 Machine Learning & Data Science Portfolio

Welcome to my Machine Learning portfolio! This repository contains a collection of my Jupyter Notebooks, showcasing my learning journey, theoretical understanding, and practical skills in Data Science.

## 🗂 Repository Structure

This repository is organized into three main sections:

### 1. Theory and Algorithms (`01_Theory_and_Algorithms/`)
Basic concepts and step-by-step breakdowns of fundamental machine learning algorithms.
* **`machine_learning_basics.ipynb`**: An introduction to core ML concepts and methodologies.
* **`linear_regression.ipynb`**: Detailed breakdown, assumptions, and implementation of Linear Regression.

### 2. Data Exploration and Visualization (`02_Data_Exploration_and_Viz/`)
Exploratory Data Analysis (EDA) and data storytelling techniques.
* **`data_tendency.ipynb`**: Statistical analysis focusing on central tendencies and data distributions.
* **`data_visualisation.ipynb`**: Creating insightful charts and graphs to uncover hidden patterns.

### 3. Applied Projects (`03_Applied_Projects/`)
End-to-end practical projects applying machine learning to various datasets.
* **`cars_analysis.ipynb`**: Exploratory analysis and modeling based on automotive data.
* **`predicting_insurance_price.ipynb`**: A regression task to predict insurance premiums based on customer features.
* **`titanic.ipynb`**: Survival prediction on the famous Titanic dataset using classification algorithms.

## 🛠️ Technologies & Libraries Used
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Machine Learning:** Scikit-Learn
* **Data Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebook
